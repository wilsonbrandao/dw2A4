Utilizando os conceitos apresentados na vídeo aula - Modelo de Objetos e Modularização, elabore dois módulos javascritp para incluir máscara, validar os campos inputs, criar classes e objetos baseados nos inputs. Os arquivos HTML, CSS e imagens utilizadas na aplicação estão localizados na pasta compacta (modules-valid-mask.rar). O aluno deve codificar os módulos usando JavaSCript e ao final salvar os arquivo em uma subpasta "A5", dentro da pasta "atividades" no repositório do github da disciplina, assim como, fazer deploy (publicar) a ferramenta web no GitHub Pages (ou quaisquer outros serviços) e enviar o link do repositório e da página da ferramenta no moddle

Link de exemplo para a inclusão de máscaras https://www.youtube.com/watch?v=r-8isv_TnVA
